# Online Learning Adaptability Prediction

## Project Overview

온라인 학습 환경에서 학생들의 적응도를 예측하고
영향 요인을 분석하기 위한 머신러닝 프로젝트

## Dataset

Source:
Kaggle - Students Adaptability Level in Online Education

## Models

- Decision Tree
- Random Forest
- KNN
- SVM
- Naive Bayes

## Results

| Model | Accuracy |
|---------|---------|
| Random Forest | 92.0% |
| Decision Tree | 91.2% |
| KNN | 84.3% |
| SVM | 78.9% |
| Naive Bayes | 62.9% |

## Key Findings

- Financial Condition
- Network Type
- Age
- Class Duration

등이 온라인 학습 적응도에 중요한 영향을 미침

## Educational Implications

온라인 교육 환경 개선을 위한
정책적 시사점을 제시함
